## Migration Guide

### From v1.0.0 to v2.0.0

1. **Update Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

2. **New Dashboard**
   ```bash
   # Old dashboard
   streamlit run enhanced_dashboard.py
   
   # New advanced dashboard
   streamlit run src/enhanced_dashboard_v4.py
   ```

3. **Configuration Changes**
   - Old: Static configuration in `config.py`
   - New: Dynamic configuration in `config.json`

4. **Docker Updates**
   - Update `docker-compose.yml` to new version
   - New service names and health checks

5. **API Changes**
   - Updated function signatures
   - New privacy engine integration
   - Enhanced logging system
